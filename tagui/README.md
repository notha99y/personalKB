# TagUI

## Useful commands
- run tagui
```bash
tagui <path to your.tag script>
```
- run headless
```bash
tagui <path to your.tag script> -h
```
- deploy to comand script
```bash
tagui <path to your.tag script> -d  
```
- interative tagui
```bash
tagui live
```
